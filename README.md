# 惜物盲盒 
這是一個旨在結合環保惜物與盲盒驚喜的文創商品平台。系統為文創賣家提供曝光與清理庫存的管道，同時讓買家能以實惠價格獲得獨特的手工藝品，達成資源再利用的共贏生態 。


## 核心功能 Core Features
```text
盲盒交易機制 (Blind Box Trading)：買家可瀏覽各店家的盲盒並下單，享受開箱驚喜；系統會根據價格自動生成對應的盲盒圖片，強化「盲盒」視覺體驗 。
賣家商店管理 (Store Management)：賣家可建立個性化商店主頁、上架文創盲盒、並透過系統化的庫存管理與自動下架功能簡化銷售流程 。
庫存與分類 (Inventory & Category)：支持「公仔、首飾、實用小物、服飾」等類別篩選與關鍵字搜尋，庫存不足時系統會自動將盲盒下架以防超賣 。
信譽評價與黑名單 (Reputation & Blacklist)：買賣雙方交易後可互留評論。賣家可針對惡意取消訂單的買家列入黑名單，確保交易環境公平透明 。
即時通知系統 (Notification System)：整合 Email 通知功能，當買家下單成功或訂單狀態異動時，系統自動發送郵件提醒賣家處理 。
```




## 系統運作流程 System Workflow

```text
Shop & Product Setup：賣家註冊並建立店鋪後即可上架商品。上架時需輸入「可能內容物」描述，系統會依價位預設盲盒封面圖 。
Order Placement：買家選購商品加入購物車，填寫寄送門市與付款方式（刷卡或貨到付款）完成下單 。
Inventory & Alert：下單完成後系統自動更新庫存。若庫存歸零，盲盒將自動從主頁移除。
Transaction Fulfillment：賣家接收 Email 通知後進行出貨，買家可隨時於訂單頁面追蹤最新的物流進度 。
Feedback Loop：交易完成後，雙方進行評價。系統根據評價建立賣家信譽指標或買家黑名單紀錄 。
```

## 系統架構 System Architecture
```text
前端: HTML5, CSS3, JavaScript (採用 Tab 切換分類設計)
後端框架: PHP 
資料庫: MySQL 
主要模組: 購物車邏輯、Email 通知模組、評價與黑名單管理系統 
```

## 檔案結構 Project Structure
```text
index.php: 買家/賣家首頁與導覽邏輯 。
account/: 處理使用者註冊、登入與個人資料修改 。
store/: 賣家商店主頁建立與管理邏輯 。
product/: 盲盒上架、修改資訊、自動下架與分類搜尋功能 。
order/: 訂單處理流程、購物車計算、郵件通知邏輯 。
admin/: 用戶意見箱回覆、黑名單列表與評價管理 。
```

## 安裝與執行 Installation & Setup
```text
資料庫配置：

匯入提供的 SQL 檔案至 MySQL 資料庫。
關聯表包含 user, store_info, product, p_order, review, blacklist 等 。

環境設定：

確保伺服器支援 PHP 運行環境（如 XAMPP 或 Apache）。
配置資料庫連線資訊於設定檔中。

啟動服務：

將專案資料夾放入伺服器根目錄，開啟瀏覽器導向 index.php 即可開始體驗 。
```

## 使用範例 Usage Guide
```text
建立商店：賣家登入後點擊「建立商店」，輸入商店名稱與品牌介紹後即可開始上架 。
購買盲盒：買家透過分類 Tab 尋找感興趣的類別（如「實用小物」），點擊加入購物車並結帳 。
查看評價：在賣家商店頁面可查閱其他買家留下的真實評價，作為購買參考 。
```
